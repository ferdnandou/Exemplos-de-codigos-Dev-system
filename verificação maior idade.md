# Verificação de Maioridade

Este script lê a idade do usuário e verifica se ele é maior de idade (18 anos ou mais).

## Código

```python
# Leitura da idade
idade = int(input("Digite a sua idade: "))

# Verificação de maioridade
if idade >= 18:
    print("Você é maior de idade.")
else:
    print("Você é menor de idade.")
